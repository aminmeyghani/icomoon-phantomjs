# icomoon-phantomjs

IcoMoon driver written in PhantomJS

## Getting Started
Install the module with: `npm install icomoon-phantomjs`

```javascript
var icomoon_phantomjs = require('icomoon-phantomjs');
icomoon_phantomjs.awesome(); // "awesome"
```
